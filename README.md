# Screen Recording Automation Tool

## 📖 Description
This project is a Python-based application that automates screen recording of online videos.  
It allows users to record a specific portion of the screen with or without audio, select quality settings, and automatically save recordings.  
The tool integrates browser automation to detect when one video ends and seamlessly start recording the next, eliminating manual effort.

## 🛠 Libraries & Tools
- **FFmpeg** – for audio/video recording and encoding
- **Playwright** – for browser automation and video playback control
- **PyQt5** – for graphical user interface (GUI)
- **OS / Pathlib** – for file management and automatic naming
- **Threading / Asyncio** – for non-blocking recording and automation
- **Logging** – for structured debugging and error tracking
